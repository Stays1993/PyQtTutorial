# PyQt6 GUI 开发示例

## 1. 项目概述

本项目是一个使用PyQt6库开发的GUI应用程序示例，包含多个窗口和界面。每个示例文件展示了如何使用PyQt6创建不同类型的窗口和界面组件。

项目的文件结构如下：

```text

    │   ├── 1.6_window_4.py
    │   ├── 1.8_WindowUI.py
    │   ├── 1.8_WindowUI.ui
```

## 3. 依赖库

- PyQt6: 用于创建GUI应用程序的Python库。
- PyQt6-Qt6: PyQt6的Qt库。
- PyQt6_sip: PyQt6的SIP绑定库。

## 4. 运行环境

- Python 3.9.18
- PyQt6 6.8.0
- PyQt6-Qt6 6.8.1
- PyQt6_sip 13.9.1

## 5. 使用方法

### 5.1. 安装依赖库：

```bash
   pip install -r requirements.txt
```
